
📊 Ecommerce Sales Insights Dashboard

An interactive Power BI dashboard built to analyze ecommerce sales performance, customer behavior, profitability, and shipping efficiency. This project helps businesses understand key performance indicators (KPIs) and make data-driven decisions.

📌 Project Overview

This dashboard provides valuable insights into ecommerce sales by analyzing:

- Total Sales
- Total Profit
- Order Quantity
- Customer Segments
- Product Categories
- Regional Performance
- Shipping Analysis
- Delivery Status
- Customer Distribution

The dashboard is designed to help stakeholders identify sales trends, profitable products, customer purchasing patterns, and operational performance.

 🛠️ Tools & Technologies
- **Power BI**
- **Microsoft Excel / CSV**
- **Power Query**
- **DAX**
- **Data Visualization**

🚀 How to Use

1. Download this repository.
2. Open **Ecommerce_Sales_Insights.pbix** using Power BI Desktop.
3. Refresh the data if required.
4. Explore the interactive dashboard using filters and slicers.

🎯 Business Insights

The dashboard helps answer questions such as:

- Which product categories generate the highest sales?
- Which regions are the most profitable?
- Which customer segment contributes the most revenue?
- How efficient is the shipping process?
- What are the sales trends over time?
